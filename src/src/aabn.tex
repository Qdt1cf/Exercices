\contenu{
\texte{
Dans un crypto-système utilisant la méthode RSA, déterminer la clef secrète 
$(\varphi(n), d)$ et le message envoyé $m \in \mathbb{Z}/n\mathbb{Z}$ 
pour les clefs publiques $(n, e)$ et les messages reçus $c = m^e \mod n$ suivants :
\begin{enumerate}
    \item $(n = 35, e = 5, c = 10)$
    \item $(n = 265, e = 139, c = 10)$
    \item $(n = 667, e = 493, c = 10)$
    \item $(n = 3599, e = 31, c = 60)$
\end{enumerate}
}
}
